# To-Do-Gether - Collaborative Todo App
## A cute shared task manager for you and your friends

### Theme
- Color Palette: Warm peachy tones (#eab676 and variations)
- Vibe: Cute, cozy, collaborative

### Setup
Run `setup.bat` to initialize the project.

### Development
1. Virtual environment is auto-activated after setup
2. Run app: `python app.py`
3. Visit: `http://localhost:5000`

### Features
- Collaborative Todo Lists
- Real-time shared viewing
- Date/Day/Time display
- User Authentication
- Edit/Delete/Reorder todos
- Cross-user todo management
